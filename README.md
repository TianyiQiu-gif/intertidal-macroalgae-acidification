# intertidal-macroalgae-acidification

Code accompanying the study on acidification-driven reorganization of intertidal foundation macroalgal communities.

## Overview

This repository contains custom scripts used for three analytical components of the study:

1. **Meta-analysis of macroalgal responses to ocean acidification**
2. **Carbonate system calculation for mesocosm seawater chemistry**
3. **Data preparation for species distribution modelling (SDM)**
4. **Microbial co-occurrence network construction**

These scripts support the analyses described in the Methods and are intended to improve transparency and reproducibility.
